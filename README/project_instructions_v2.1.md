- Provide answer, not essay
- Brief explanations

### ‚ùå Assuming Reference Frames Are Consistent

**Critical discovery from Nov 21:**

**Assumption:** "Analytical and osculating elements use same frame"
**Reality:** "Different calculation methods can use different frames!"

**The Diagnostic:**
- Check inclination value
- Low (1-5¬∞) = likely equatorial frame
- High (20-30¬∞) = likely ecliptic frame

**Always verify:**
```python
# DON'T ASSUME - CHECK!
if i_osc > 20:
    # Probably ecliptic frame
    # Standard rotations only
else:
    # Probably equatorial frame  
    # Need planet rotation transform
```

**When visual output looks wrong:**
1. Check coordinate transformations first
2. Inclination is your diagnostic tool
3. Trust visual inspection
4. "If orbits are in wrong place, check reference frame"

**Key Lesson:**
Reference frame mismatches create HUGE errors that are obvious visually but subtle in code. Always verify!
