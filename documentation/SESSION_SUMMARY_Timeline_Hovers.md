## Angled Arrow Technique (Bonus)

**Tony asked**: "How did you do the angled arrow?"

**Answer**: `ax` and `ay` parameters create vector from text to point
```python
ax=40   # Go 40 pixels right
ay=-60  # Go 60 pixels up (negative = up)
# Result: Diagonal arrow ↗️
```

**Other arrows:**
- `ax=0, ay=-40` → Straight down ⬇️
- `ax=40, ay=0` → Straight right ➡️
- `ax=-40, ay=40` → Diagonal down-left ↙️

**Experiment freely** - just pixel offsets!

## Technical Notes

### Color Coding Strategy:
- **Timeline**: Crimson red (time markers)
- **Warm events**: Orange (MWP)
- **Cool events**: Blue (LIA), Turquoise (YD)
- **Recent events**: Green (Holocene start)
- **Modern**: Red (Anthropocene)
- **Deep time "?"**: Period-specific colors

### Arrow Directions:
- **Warming**: Arrow points UP (MWP)
- **Cooling**: Arrow points DOWN (LIA, YD)
- **Recent/cluttered**: Angled arrows (Anthropocene)
- **Older events**: No arrows, just "?" with hover

### Positioning Strategy:
- **Timeline**: Top of plot (y=1.00 or y=0.905)
- **Recent events**: Avoid overlap with arrows
- **Deep time events**: "?" at event location
- **Log scale considered**: Recent events cluster (normal!)
